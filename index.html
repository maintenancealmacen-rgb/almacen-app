Configuración

// API Google Sheets

// Cómo configurar (paso a paso)

1
Ve a console.cloud.google.com e inicia sesión con tu cuenta de Google
2
Crea un proyecto nuevo (o usa uno existente). Nombre: Almacen App
3
Ve a APIs y servicios → Biblioteca y activa Google Sheets API
4
Ve a APIs y servicios → Credenciales → Crear credencial → Clave de API. Copia la clave y pégala arriba
5
En tu Google Sheets, ve a Compartir y cambia a "Cualquier persona con el enlace puede editar"
6
El ID del spreadsheet es la parte de la URL entre /d/ y /edit
7
Guarda la configuración y prueba desde la pantalla de inicio

// Instalar como App

En Chrome Android: pulsa el menú ⋮ → "Añadir a pantalla de inicio".

La app funcionará sin conexión y aparecerá como una app nativa.

// Email de alertas

Se enviará un aviso automático cuando el stock quede por debajo del mínimo.
El email se envía desde el propio Google Script — no necesita configuración adicional.
